# IT342 G6 Tan Lab 1

## Project Overview
This project contains a comprehensive web application with backend, frontend, and mobile components.

## Project Structure
```
IT342_G6_Tan_Lab1/
├─ /web              # Web frontend application
├─ /backend          # Backend API (Spring Boot)
├─ /mobile           # Mobile application (placeholder)
├─ /docs             # Documentation
├─ README.md         # This file
└─ TASK_CHECKLIST.md # Task completion checklist
```

## Setup Instructions

### Backend Setup
1. Navigate to the `/backend` directory
2. Run `mvn clean install` to install dependencies
3. Run `mvn spring-boot:run` to start the application

### Web Frontend Setup
1. Navigate to the `/web` directory
2. Follow the setup instructions in the web folder

### Mobile Setup
1. Navigate to the `/mobile` directory
2. Follow the setup instructions in the mobile folder
